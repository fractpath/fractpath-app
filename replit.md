# FractPath

## Overview
FractPath is a static website placeholder (Sprint 0). The app code will be developed in Sprint 1.

## Project Structure
- `index.html` - Main HTML file served as the static site
- `docs/` - Documentation directory
- `sprints/` - Sprint planning documentation

## Running the Project
The site is served using `npx serve` on port 5000.

## Deployment
This is a static site that serves HTML files directly.
